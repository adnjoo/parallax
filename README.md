# Parallax Demo

A 2D scrolling game demo built with Godot 4.5 featuring parallax backgrounds and a spaceship player character.

![Screenshot](assets/misc/screen.png)

## Features

- **Parallax Scrolling**: Multiple background layers with different scroll speeds
- **Player Movement**: Vertical movement with horizontal auto-scroll
- **Multiple Environments**: 
  - City Night
  - City Destroyed
  - Rural Area

## Controls

- **Up/Down Arrow Keys**: Move the spaceship vertically
- The spaceship automatically scrolls horizontally

## Setup

1. Open the project in Godot 4.5 or later
2. Run the main scene (`src/main.tscn`)

## Project Structure

```
parallax/
├── assets/          # Sprites and parallax layers
├── src/             # Game scenes and scripts
└── project.godot    # Godot project configuration
```
